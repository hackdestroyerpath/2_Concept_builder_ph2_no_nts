# Task template

[← Templates](../README.md) | [State](item_state.md) | [Registry](page_registry.jsonl)

`Templates/task/` хранит базовые артефакты управляемой задачи.

## Файлы

| Файл | Назначение |
|---|---|
| `item_state.md` | Состояние задачи. |
| `question_answer.md` | Уточнения. |
| `requirements.md` | Требования. |
| `solution.md` | Решение. |
| `contract.md` | Контракт выполнения. |
| `report.md` | Отчёт. |
| `linked_files.md` | Связанные файлы. |
| `page_registry.jsonl` | Локальный реестр шаблона. |

## Правило

Шаблон используется через `issue_lifecycle`, `requirements_protocol`, `issue_execution` и `template_validation`.
