# Templates

[← К точке входа](../README.md)

`Templates/` хранит production-шаблоны, которые используются протоколами `Concept Builder`.

## Доступные шаблоны

| Шаблон | Назначение |
|---|---|
| [`concept/`](concept/README.md) | Базовая структура новой концепции для `Execution Mode`. |

## Правила

1. Шаблон не является готовой концепцией.
2. При создании концепции значения-заглушки заменяются конкретными данными.
3. Изменение шаблонов выполняется только через `Service Mode`.
4. Копирование шаблона в `Concepts/<concept_id>/` выполняется через `Protocols/execution_bootstrap.md`.
