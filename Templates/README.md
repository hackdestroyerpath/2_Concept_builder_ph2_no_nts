# Templates

[← К точке входа](../README.md)

`Templates/` хранит production-шаблоны системы.

## Шаблоны

| Шаблон | Назначение |
|---|---|
| [`concept/`](concept/README.md) | Структура новой концепции. |
| [`task/`](task/README.md) | Артефакты управляемой задачи. |

## Правила

1. Шаблон не является готовым объектом работы.
2. Значения-заглушки заменяются при создании объекта.
3. Шаблоны меняются только через `Service Mode`.
4. Проверка выполняется через `Protocols/template_validation.md`.
