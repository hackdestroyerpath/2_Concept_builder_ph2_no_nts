# Маршрутизация режимов

[← Реестр протоколов](protocol_index.md)

## Назначение

Файл описывает выбор между `Service Mode` и `Execution Mode`.

## Правило

`Service Mode` используется для протоколов, инструкций, state и реестров.

`Execution Mode` используется для выбранной концепции внутри `Concepts/`.

Если задача смешивает режимы, агент сначала разделяет её на части и работает только в разрешённой области.
